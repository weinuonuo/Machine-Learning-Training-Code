题目描述
> 经典程序设计问题：找第n个默尼森数。P是素数且M也是素数，并且满足等式M=2**P-1，则称M为默尼森数。例如，P=5，M=2**P-1=31，5和31都是素数，因此31是默尼森数。

> 输入格式:按提示用input()函数输入

> 输出格式：int类型

> 输入样例：4

> 输出样例：127
